# 🌌 Universo – Bibliothèque en ligne pour auteurs

**Universo** est une plateforme web conçue pour les auteurs et passionnés de lecture. Chaque utilisateur peut :

- Publier ses histoires en épisodes (comme sur Wattpad)
- Vendre ses livres et ebooks dans une boutique en ligne
- Lire des articles de blog littéraires
- Suivre des cours pour progresser dans l’écriture

## 🚀 Fonctionnalités principales
- Espace auteur personnalisé
- Interface intuitive
- Page d’accueil, blog, boutique, e-learning
- Formulaire de connexion/inscription sur une seule page

## 🛠️ Technologies utilisées
- React.js
- React Router
- Tailwind CSS (ou autre framework CSS)
- (Possibilité d’intégrer Firebase ou Node.js plus tard)
